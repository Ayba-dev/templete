/\*\*

- Объект `cell` представляет собой маппинг типов ячеек к их React-компонентам.
-
- **Описание:**
- Каждый ключ объекта соответствует определённому типу ячейки, а значение –
- компоненту, отвечающему за отображение данной ячейки. После добавления нового ключа, вы можете использовать его в массиве columns таблицы, в поле type
-
- **Тип:** `Record<string, FC>`
-
- **Пример использования:**
- {
- title: 'Время проведения',
- dataIndex: 'created_at',
- type: 'date',
- }
